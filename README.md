# 🖐️ Hand Gesture Controlled Mouse (Python + OpenCV + MediaPipe)

This project implements a **virtual mouse using hand gestures** with the help of **Computer Vision**.  
It allows users to control the mouse cursor using their **index finger movement** and perform clicks using **thumb + index finger gestures**.

This system uses a webcam to detect hand landmarks and converts gestures into mouse actions in real-time.

---

## 🚀 Features

✔ Cursor movement using index finger  
✔ Left click using thumb + index finger gesture  
✔ Smooth cursor movement  
✔ Real-time hand tracking  
✔ Works with webcam  
✔ macOS compatible  

---

## 🧠 Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Computer Vision
